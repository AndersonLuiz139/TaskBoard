# ✅ TaskBoard

O **TaskBoard** é uma aplicação de gerenciamento de tarefas desenvolvida com **HTML, CSS e JavaScript puro**.  
Com ele, você pode organizar suas tarefas de forma prática, marcar atividades concluídas, editar itens e manter tudo salvo no navegador com `localStorage`.

## ✨ Funcionalidades

- Adicionar novas tarefas
- Editar tarefas existentes
- Marcar tarefas como concluídas
- Desfazer tarefas concluídas
- Apagar tarefas
- Filtrar por:
  - Todas
  - Pendentes
  - Concluídas
- Limpar tarefas concluídas
- Contador total de tarefas
- Salvamento automático no `localStorage`
- Interface responsiva

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido para praticar conceitos essenciais do front-end, como:

- Manipulação do DOM
- Eventos com JavaScript
- Criação dinâmica de elementos
- Armazenamento local no navegador
- Responsividade e organização de interface
